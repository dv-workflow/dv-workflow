# dw-kit-audit — Quarterly Evolution Review

Nhìn lại: **$ARGUMENTS ngày** (mặc định 90 nếu không truyền)

> Maintainer-only skill. Dùng bởi TechLead trong dw-kit repo.
> Không ship theo npm (xem .npmignore).

---

## Bước 1: Thu Thập Issues

```bash
# Closed issues trong N ngày qua
gh issue list \
  --repo dv-workflow/dv-workflow \
  --state closed \
  --limit 200 \
  --json number,title,labels,closedAt,body \
  | sort by closedAt desc
```

Cũng lấy open issues có label `needs-evolve-review` (tồn đọng chưa xử lý):
```bash
gh issue list \
  --repo dv-workflow/dv-workflow \
  --state open \
  --label "needs-evolve-review" \
  --json number,title,labels,createdAt
```

## Bước 2: Group & Pattern Detection

Group issues theo `component` × `type`:

```
component: hooks
  bug:       [#12, #18, #24]  → 3 issues
  friction:  [#9]             → 1 issue

component: workflow
  friction:  [#7, #15, #22]   → 3 issues
  gap:       [#11]            → 1 issue

component: skills
  suggestion: [#5, #8]       → 2 issues
...
```

**Pattern threshold:**
- **3+ issues cùng component+type** → flag là pattern, đánh giá sâu
- **1-2 issues** → ghi nhận, không action ngay
- **0 issues** cho một component trong kỳ → positive signal (ghi nhận)

---

## Bước 3: Đánh Giá Pattern

Với mỗi pattern được flag (≥3 issues), phân tích:

**Câu hỏi cần trả lời:**
1. Root cause chung là gì? (không phải symptom)
2. Fix từng issue riêng lẻ có đủ không, hay cần thay đổi rule/design?
3. Pattern này cho thấy dw đang thiếu gì, hay đang có gì thừa?

**Verdict options:**
- `rule-change`: cần sửa rule/behavior hiện tại
- `new-feature`: cần thêm tính năng mới
- `docs-fix`: vấn đề là documentation không rõ, không phải behavior
- `retire`: rule/feature nên bị loại bỏ (chỉ khi usage < 30%)
- `no-action`: issues đã resolved đúng cách, không cần thay đổi systemic

## Bước 6: Tự Động Flag Tồn Đọng

Nếu có issues `needs-evolve-review` quá 14 ngày chưa xử lý:

```bash
# Comment nhắc nhở lên issue
gh issue comment [N] \
  --repo dv-workflow/dv-workflow \
  --body "⏰ Issue này đã chờ review 14+ ngày. /dw-kit-evolve #[N] để xử lý."
```
